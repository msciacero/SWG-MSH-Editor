SWG-MSH-Editor
==============

1) ABOUT
Can import msh files into 3D Studio Max and can convert 3D studio max files into MSH files.

2) INSTALL
Download the scripts and run script in 3D Studio Max. Not a plugin so it can work with versions 2006 and later
